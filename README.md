A beginner-friendly Spring Boot REST API project to practice controllers, services, and basic CRUD operations.

# Lost & Found REST API

A simple Spring Boot project built as a practice/revision project.  
The app allows adding, viewing, and deleting items stored in memory.

## Features
- Spring Boot basics: `@Component`, `@Autowired`  
- Separation of Service and Controller  
- REST endpoints:
  - `GET /items` – view all items  
  - `GET /items/add?item=...` – add an item  
  - `GET /items/delete?item=...` – delete an item
